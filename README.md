# Text Mining Project

# Project background

The project is the outcome of an internship with an energy company that wanted to utilise their data from customer service logs in order to obtain insights. It was created as a proof of concept over a two-week internship.

There is a demonstration video attached.

# Tech stack used
The data was stored in ElasticSearch.
The front-end was implemented using Streamlit, which is a framework on top of Python.
